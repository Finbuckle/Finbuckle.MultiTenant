## About MultiTenant

MultiTenant is open source multi-tenancy middleware library for .NET created and maintained by [Finbuckle LLC]
(https://www.finbuckle.com). It enables tenant resolution, per-tenant app behavior, and per-tenant data isolation.

See [https://www.finbuckle.com/MultiTenant](https://www.finbuckle.com/multitenant) for more details and documentation.

See the [release history](https://www.finbuckle.com/MultiTenant/Docs/History) for all release details.
